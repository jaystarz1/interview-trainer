Why Microsoft is Struggling

  • Nadella admitted: "Integrations don't really work... not smart enough yet"
  • Architecture problem: Built as thin layer over Office, not rethinking workflows
  • Data chaos: Copilot surfaces files users shouldn't see (over-permissioning)
  • 6 different Copilots: M365, Windows, Edge, GitHub, Dynamics, Security - confusing
  • US Congress banned it: Data security and accuracy concerns
  • Lawsuit: Microsoft sued for misleading 2.7M users into subscriptions

What Users Actually Say

  • "Practically useless" for document editing, Excel formulas, PPT
  • Gives instructions instead of doing - "here's how to..." when users wanted it DONE
  • Hallucinations: Fabricated dates, wrong facts, invented meetings
  • Slow: Response times frustrate users used to instant tools
  • Generic outputs: Bland corporate text nobody would actually send
  • "$30/month for this?" - 50% of enterprises can't justify the cost

Deeper Technical Issues

  • Context window limits: Can't process long documents well - truncates or misses key parts
  • No memory across sessions: Every conversation starts fresh (until late 2025 update)
  • SharePoint integration weak: Poor at finding files in complex folder structures
  • Excel is the worst: Formula generation unreliable, pivot tables broken
  • Formatting loss: Summaries strip important structure from documents
  • Teams transcription quality: Meeting summaries miss nuance, attribute wrong speakers
  • Prompt sensitivity: Small wording changes produce wildly different results

What This Means for CFMWS

  • 4,000 licenses = 4,000 gym memberships collecting dust
  • Gray AI happening now: People using ChatGPT because Copilot disappoints
  • First impressions bad: Most tried it, got garbage, gave up
  • Power users exist: Someone has figured it out - find them
  • Expectations need resetting: It's an assistant, not a replacement
  • Role-specific prompts are key: Facility manager ≠ HR ≠ Finance

"You gave everyone Copilot. That's a gym membership. I'm the personal trainer who shows up and says 'let me show you exercises that actually work for YOUR body.'"

What Actually Works

  • Meeting summaries - genuinely saves time (auto-recap)
  • Email drafts - first pass on routine messages
  • Document summarization - long reports → key points
  • Brainstorming partner - idea generation, not final copy
  • Search/Recall - "find emails about X from last month"
  • Translation/tone adjustment - reformulate for different audiences

Where It Fails

Complex edits, precise formatting, tasks requiring judgment, anything needing accuracy, real-time data. Set expectations early.

The Champions Model

  • 10,000 internal "Champs" - Microsoft's own approach
  • Train-the-trainer: Enable champs to run lunch-and-learns
  • Community calls: Showcase wins, share best practices
  • Gamification: +24% engagement, +50% productivity
  • Camp Copilot: 3-week intensive adoption sprints
  • Peer influence: "Colleagues > IT handouts"

"Peer influence scales further and faster than policy alone. Employees show more interest in content their colleagues create."

RTF Framework for Prompts

Most users fail because they don't know how to ask:

  • R - Role: "Acting as an experienced project manager..."
  • T - Task: "Create a status update for executive leadership..."
  • F - Format: "As a 3-paragraph email with bullet points"

Prompts That Actually Work

"Summarize this into a 300-word executive summary focusing on outcomes, risks, and next steps."
"Rewrite this email for a senior executive. Keep it under 150 words. Professional but warm."

Your Play for CFMWS

  • Acknowledge reality: "Copilot out of the box disappoints"
  • Reframe: "But with training, it saves 30+ min/day"
  • Find gray users: Someone is making it work - make them heroes
  • Role-specific training: Different prompts for different jobs
  • Prompt libraries: Copy-paste prompts that actually work
  • Super User network: Peer trainers in every division
